# 🚀 AI PDF 精密蓋章工具 (Big Font & Auto-Currency Edition)

這是一個純前端（Client-side）的 PDF 處理工具，專為需要快速在發票（Invoice）或文件上蓋章並填寫驗證資訊的工作流程而設計。

## ✨ 主要功能
- **精密坐標對齊**：解決 PDF 與 Canvas 坐標系不一致導致的走位問題，拖到哪裡，印到哪裡。
- **大字體設計**：針對打印需求優化，標籤與填寫數值皆採用加大字體（12px/15px Bold）。
- **智能自動填寫**：自動識別 PDF 內容（特別針對 Cardinal Health 格式），擷取總金額。
- **自動金額格式化**：手動輸入數字時，系統會自動補上 `$` 符號。
- **動態檔名輸出**：下載時自動將原檔名後綴改為 `_verified.pdf`。
- **零伺服器處理**：所有 PDF 處理均在瀏覽器完成，文件不會上傳至任何伺服器，確保私隱。

## 🛠️ 技術棧
- [pdf-lib](https://pdf-lib.js.org/) - 用於修改 PDF 頁面內容。
- [pdf.js](https://mozilla.github.io/pdf.js/) - 用於在瀏覽器渲染 PDF 預覽。

## 🚀 如何部署 (Deploy)

### 方案 A：GitHub Pages (推薦)
1. 在 GitHub 建立一個新倉庫 (Repository)。
2. 將 `index.html` 上傳至倉庫。
3. 前往 **Settings > Pages**。
4. 在 **Build and deployment** 下選擇 `main` 分支並點擊 **Save**。
5. 等待約 1 分鐘即可通過生成的網址訪問。

### 方案 B：本地使用
直接用瀏覽器打開 `index.html` 即可運行，無需安裝任何環境。

## 📖 使用說明
1. **上傳**：將 PDF 檔案拖入灰色虛線區域，或點擊「選擇 PDF」。
2. **編輯**：在右側面板修改日期、編號或金額（金額會自動加 $）。
3. **定位**：在預覽圖中直接拖動「印章」到合適位置。
4. **下載**：點擊「儲存並下載」，系統將生成帶有印章的新 PDF。

## 🔒 隱私聲明
本工具不設後端伺服器。所有 PDF 操作均在用戶本地瀏覽器的記憶體中完成，您的財務文件絕對安全。
